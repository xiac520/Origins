Origins

通过 API 获取任何页面的内容（JSON/P 或原始内容），并避免同源策略问题。

示例

从 https://example.org 获取数据：

fetch(`https://origins.dnscron.com/get?url=${encodeURIComponent('https://example.org')}`)
  .then(response => {
    if (response.ok) return response.json();
    throw new Error('网络响应不正确。');
  })
  .then(data => console.log(data.contents));

或者使用 jQuery

$.getJSON('https://origins.dnscron.com/get?url=' + encodeURIComponent('https://example.org'), function (data) {
  alert(data.contents);
});

选项

charset

描述: 设置响应字符编码 (charset)
示例: https://origins.dnscron.com/get?charset=ISO-8859-1&url=https://example.org/

raw

描述: 获取原始内容 (CORS)
示例: https://origins.dnscron.com/raw?url=https://example.org/

callback

描述: 获取 JSONP 响应
示例: https://origins.dnscron.com/get?callback=myFunc&url=https://example.org/
